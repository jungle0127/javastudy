## MD - Message Digest
MD5是继MD2，MD4算法发展而来，算法复杂度和安全强度大大提高。MD算法的最终结果都是一个128位的消息摘要。
MD5算法执行效率略次于MD4，安全性方面略胜一筹。
MD5算法暴露出的漏洞越来越多，已不再适合安全要求较高的场合使用。
## SHA - Secure Hash Algorithm
五种SHA算法：
SHA-1（=SHA）
SHA-2: SHA-256,SHA-224,SHA-384,SHA-512
